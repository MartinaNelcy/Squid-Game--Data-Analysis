# Squid Game Data Analysis

## About the Project
This project is inspired by the Squid Game series and focuses on analyzing player survival patterns. I created a dataset and explored how different games affect survival and outcomes.

The goal was to practice data analysis using Python and understand how to turn raw data into meaningful insights.

---

## What I Did
* Created a dataset of 15,000 players
* Included features like age, gender, game played, and survival status
* Analyzed which games are more dangerous
* Visualized the results using charts

## Tools Used
* Python
* Pandas
* Matplotlib
* Jupyter Notebook

---

## About the Dataset
The dataset includes:
* Player ID
* Age
* Gender
* Game name
* Status (Alive / Dead)
* Money won
* Round number---

## Analysis
I explored:
* Number of players who survived vs died
* Which game had the highest number of deaths
* How outcomes vary across different games

---

## Visualizations
* Pie chart showing survival distribution
* Bar chart showing the most dangerous games

---

## Key Findings
* Glass Bridge has the highest number of deaths
* Tug of War also shows a relatively high failure rate
* Overall survival is almost balanced
* Only surviving players earn money
